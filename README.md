# Food Zero 🍽️

A modern and elegant restaurant landing page built using **HTML**, **CSS**, and **Bootstrap**.

This project showcases a premium food website UI with multiple responsive sections including hero banner, menu, blogs, reservation form, testimonials, and footer.

---

## 🚀 Features

- Fully Responsive Design
- Modern Restaurant UI
- Bootstrap Layout System
- Clean & Organized Code Structure
- Custom Styling with CSS
- Reservation Form Section
- Blog & Testimonial Sections
- Professional Footer Design
- Reusable Utility Classes

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Google / Offline Fonts
